# 개발 가이드 (dev)

## 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행 (Hot reload)
npm run dev

# 프로덕션 빌드
npm run build

# 빌드 결과물 로컬 미리보기
npm run preview
```

## 기술 스택
- React 18, TypeScript
- Vite (빌드·개발 서버)
- sql-formatter (SQL 포매팅)
