## 🐞 이슈 및 해결

### @apply 참조 에러
**문제**: CSS 파일 간 `@apply` 참조 불가  
**원인**: `@apply`는 같은 파일 내 클래스만 참조 가능  
**해결**: 직접 스타일 적용으로 변경, 기능은 그대로 유지

### Tailwind 경고
**현상**: `@tailwind`, `@apply` 경고 메시지  
**원인**: Tailwind CSS 설정 관련 lint 경고  
**영향**: 기능에는 문제 없음
